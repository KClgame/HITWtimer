# ==================== 陷阱配置模板 (Trap Configuration Template) ====================

# 说明 / Instructions:
# 1. 每个陷阱必须以 name= 开头 / Every trap must start with name=
# 2. 所有参数都可以按需填写，不需要的可以删除 / All fields are optional except name and events
# 3. events 阶段从上到下依次执行 / Events are executed from top to bottom in order
#
# 【参数详解 / Parameter Reference】
# name: 陷阱的唯一标识，用于命令和内部匹配 (必填)
# enabled: 是否启用该陷阱 (true/false，默认 true)
# source: 检测来源 SUBTITLE / CHAT / BOTH (默认 SUBTITLE)
# subtitle_preparation: 单个陷阱的准备阶段。false 表示严格无准备时间 (不回退到列表/全局)
# start_sound / end_sound: 阶段开始/结束播放的音效
# preparation_time: 该陷阱的准备时长 (秒)
# preparation_color: 准备阶段HUD颜色 (0xRRGGBB 或 #RRGGBB)
# pattern: 检测匹配的字幕/聊天文本 (必填，支持 a|b 作为多个备选)
# match_mode: CONTAINS / EQUALS / STARTS_WITH / REGEX (默认 CONTAINS)
# case_sensitive: false (默认不区分大小写)
# events: 阶段列表，每行一个 label|time|color?|sound? (推荐格式)
#
# 英文版说明 (English):
# Each trap starts with name= . All other fields optional except events.
# subtitle_preparation=false on a trap means NO preparation (strict, ignores other levels).
# Use /hitwtimer reload after editing files.
